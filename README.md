# Vampirism KubeJS Integrations

This repository contains KubeJS scripts created to enhance and balance the experience of servers using the [Vampirism](https://www.curseforge.com/minecraft/mc-mods/vampirism-become-a-vampire) mod, by integrating it with other mods and plugins.

## Included Scripts

* **[AutoTeams](./vampirism-autoteams/)**: Automatically assigns LuckPerms groups and CustomNPCs factions based on the player's Vampirism/Werewolves faction.
* **[Toxony Disable Mutagen](./toxony-disable-mutagen/)**: Blocks and removes specific mutations from the Toxony mod for Vampires and Werewolves, ensuring server balance.
