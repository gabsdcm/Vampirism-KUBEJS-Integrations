# Vampirism KubeJS Integrations

KubeJS scripts created to balance and automate processes of servers using that use [Vampirism](https://www.curseforge.com/minecraft/mc-mods/vampirism-become-a-vampire) mod, by integrating it with other mods and plugins.

## Included Scripts

* **[AutoTeams](./vampirism-autoteams/)**: Automatically assigns LuckPerms groups and CustomNPCs factions based on the player's Vampirism/Werewolves faction.
* **[Toxony Disable Mutagen](./toxony-disable-mutagen/)**: Blocks and removes specific mutations from the Toxony mod for Vampires and Werewolves, for balance reasons.

## How to use
* Download **[KubeJS](https://www.curseforge.com/minecraft/mc-mods/kubejs/)** and install it in your server
* Put any of these scripts inside 'server-scripts' in the kubejs directory on the server folder
